虚拟环境搭建————

    下载virtualenv——
        pip install virtualenv

    创建虚拟环境——
        virtualenv diango_basic_venv

    切换虚拟环境——
        django_basic_venv\Scripts\activate

    在虚拟环境下安装开发软件——
        pip install django(软件名)==1.9.5(版本)

    查看目录下的软件安装情况——
        pip freeze

    退出虚拟环境——
        deactivate
